X-LDSC

X-LDSC (trans-ethnic stratified LD score regression) is a software package for stratifying squared trans-ethnic genetic correlation from GWAS summary statistics. The latest version is version 0.1-beta. Baseline-LD-X annotations and LD scores can be downloaded here.

Software requirement

  • Python 2.7 (sorry!)
  • NumPy 1.15+
  • Pandas 0.23.4+
  • PySnpTools 0.3+
  • SciPy 1.1.0+
  • StatsModels 0.9.0+
